# 3-Phase Contactor 3D Visualization
An interactive web-based teaching dashboard for visualizing a 3-phase magnetic contactor in 3D. The project uses Three.js through React Three Fiber and is designed for classroom demonstrations, technical training, and self-study on industrial electrical control components.
Live demo:
https://mrcong1.github.io/contactor-3d-viewer/
Overview
This application displays a 3D GLB model of a 3-phase contactor inside a visual dashboard. Students can rotate, zoom, and inspect the model directly in the browser while reading structured learning notes about the main contactor parts.
The interface focuses on:
Main power contacts: L1, L2, L3 and T1, T2, T3
Auxiliary contacts: normally open and normally closed contacts
Coil supply terminals: A1 and A2
Typical contactor functions in motor control circuits
Features
Interactive 3D contactor model rendered with Three.js
Mouse-controlled rotation, zoom, and inspection
Educational dashboard layout for classroom visualization
English learning content for 3-phase contactor components
Responsive web deployment through GitHub Pages
Static build output that can be uploaded directly to a GitHub repository
